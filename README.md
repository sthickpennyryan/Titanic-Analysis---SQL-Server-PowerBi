Titanic Data Pipeline & Analysis Project

This project demonstrates practical skills in:

Data engineering fundamentals (SQL Server)
Data cleaning and validation (SQL Server)
Analytical thinking (SQL Server + PowerBi)
Data visualisation (PowerBi with DAX)

Project Overview:
This project uses the following Kaggle dataset: https://www.kaggle.com/competitions/titanic/data. 

While the competition is for machine learning, my focus was on:
  Data cleaning and validation
  Exploratory data analysis (EDA)
  Dashboarding and storytelling through visualisation

My initial EDA focused on understanding the datasets and their data quality. 
  From the initial EDA I identified nulls,  missing values and understood the table relationships and future joins. 

Further analysis included and answered the questions found here:
https://github.com/sthickpennyryan/Titanic-Analysis---SQL-Server-PowerBi/blob/main/Analysis/Titanic%20Questions%20and%20Answers.docx

Questions were answered and explained using visuals. All dashboards, including the primary Dashboard as well as the main file can be found here:
https://github.com/sthickpennyryan/Titanic-Analysis---SQL-Server-PowerBi/tree/main/Dashboards%20(PowerBi)

Key Insights:
  Female passengers had significantly higher survival rates
  First-class passengers were more likely to survive
  Younger passengers had slightly higher survival probability
